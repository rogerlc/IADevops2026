Arquitectura Medallion
la arquitectura medallon es un modelo para la organización de datos esto nos ayuda a tener una forma mucho mas  limpia, entendible y eficaz para con trabajar con ellos.
La forma de organizar los datos siguen un flujo de tres etapas.
Bronce, Plata, Oro.
Bronce(Bronze) :
Los datos llegan exactamente como se reciben sin ninguna modificación (se guarda  una copia).
Plata (Silver):
Es la etapa en la cual los datos se corrigen se organizan, se complementan y se eliminan errores.
Oro (Gold):
Ya los datos están listos para ser utilizados en el negocio.
